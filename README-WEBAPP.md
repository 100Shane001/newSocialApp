# My Social Media App

This is a cyberpunk-themed social media app to showcase modern web dev skills.

Built with: React, Vite, TypeScript, Node.js

## Description
--AppName-- is a futuristic social media interface designed to showcase interactive UI elements, state management, and full-stack architecture skills. It's **not meant to be a production platform**, but a playground for experimenting with React, animations, and styling.

## Features
- User-friendly neon/cyberpunk interface
- Post creation and deletion
- Liking and Disliking posts
- Sharing posts
- Account creation
- Follow system
- Comments
- Profile pages
